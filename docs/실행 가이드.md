# 장소 예약 시스템 실행 가이드

## 사전 요구사항

- **Node.js**: v18 이상 권장
- **npm**: Node.js와 함께 설치됨

## 실행 방법

### 1. 프로젝트 디렉토리로 이동

```bash
cd frontend
```

### 2. 의존성 설치

최초 실행 시 또는 package.json이 변경된 경우 실행합니다.

```bash
npm install
```

### 3. 개발 서버 실행

```bash
npm run dev
```

### 4. 브라우저에서 접속

개발 서버가 실행되면 아래 주소로 접속합니다.

- **URL**: http://localhost:5173

## 기타 명령어

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | 개발 서버 실행 (Hot Reload 지원) |
| `npm run build` | 프로덕션 빌드 생성 |
| `npm run preview` | 빌드된 결과물 미리보기 |
| `npm run lint` | ESLint 코드 검사 |

## 문제 해결

### 포트 충돌 시

5173 포트가 이미 사용 중인 경우, Vite가 자동으로 다른 포트를 할당합니다.
터미널에 표시되는 URL을 확인하세요.

### 의존성 오류 시

```bash
rm -rf node_modules
npm install
```

### 캐시 문제 시

브라우저의 localStorage를 초기화하면 모든 데이터가 삭제됩니다.
개발자 도구 > Application > Local Storage에서 삭제할 수 있습니다.
