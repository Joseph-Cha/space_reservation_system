# 목표
권한이 있는 사람만 장소를 예약할 수 있는 시스템 개발

1. 세부 기능
    * 기능 명세
        * Flow 정의
            1. 캘린더 화면
            2. 날짜 클릭
            3. 장소 리스트 및 시간 테이블 화면으로 이동
                | 장소/ 시간 | vision on | 월드비전홀 | 순보금자리 | 푸른초장 | 쉴만한 물가 | 넘치는 잔 | Vision Factory 1 | Vision Factory 2 | Vision Factory 3 | Vision Factory 4 | Vision Factory 5 |
                | | | | | | | | | | | 
                | 07:00 ~ 07:30 | 
                | 07:00 ~ 07:30 | 
                ....
                | 21:30 ~ 22:00 | 
            4. 시간 클릭
            5. 예약 상세 정보 기입 팝업
                * 소속
                * 이름
                * 목적
                * 이용 시간(30분 단위로 선택 가능)
                    * 시작 시간: 선택한 시간이 기본 값이지만 추후 변경 가능
                    * 종료 시간: 선탁한 시간이 기본 값이지만 추후 변경 가능
