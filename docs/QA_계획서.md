# 장소 예약 시스템 QA 계획서

## 1. 개요

### 1.1 문서 목적
본 문서는 장소 예약 시스템의 품질 보증(QA) 테스트 계획을 정의합니다.

### 1.2 테스트 대상
- **시스템명**: 장소 예약 시스템 (Space Reservation System)
- **버전**: 1.0
- **테스트 일자**: 2026년 1월
- **테스트 환경**: 웹 브라우저 (Chrome, Safari, Mobile)

### 1.3 테스트 범위
| 구분 | 포함 | 제외 |
|------|------|------|
| 기능 테스트 | 모든 사용자 기능 | 백엔드 단위 테스트 |
| UI/UX 테스트 | 반응형 디자인, 접근성 | 성능 벤치마크 |
| 통합 테스트 | Supabase DB 연동 | 부하 테스트 |
