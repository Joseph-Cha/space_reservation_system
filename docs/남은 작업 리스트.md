# 남은 작업 리스트

1. 회원 가입 관리 -> 각 소속별 계정을 발급
2. 장소 예약 시스템 화면 -> 달력의 크기 조절(지금 너무 큼)
3. 소속 / 부서를 drop down menu로 변경
4. 예약 정보 입력 화면에서 취소와 예약하기 버튼 크기 동일하게 변경
5. 장소 예약 > 시간 / 장소 > 해당 열 데이터 표시를 30분 범위 시간으로 변경(ex. 07:00 ~ 07:30)
6. 장소 예약에서 예약이 완료될 경우 소속에 따라 채우는 색상을 변경
7. 본인이 예약한 장소에 대해서는 본인만 예약 정보를 수정할 수 있도록 변경
8. 관리자 권한이 있는 사용자는 모든 예약에 대한 정보를 수정할 수 있도록 변경
9. 본인이 예약한 정보를 한눈에 확인할 수 있는 별도의 페이지 생성

### 소속 정의
* 교역자
* 비전브릿지
* CAM
* 프뉴마
* 가스펠
* 카리스
* 기타
